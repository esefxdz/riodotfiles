# qBittorrent Themes

two themes in `themes/custom/`:

- `DarkRed.qbtheme` — matches the rio aesthetic. use this one.
- `ICEBERG.qbtheme` — iceberg style alternative, if you ever feel like it

## applying

the install script drops these into `~/.local/share/qBittorrent/themes/`.

to activate:
1. open qBittorrent
2. Tools → Preferences → Behavior → Interface
3. tick **Use custom UI theme**
4. click the folder icon and pick `DarkRed.qbtheme`
5. restart qBittorrent
